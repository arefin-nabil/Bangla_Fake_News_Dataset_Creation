# Bangla Fake News Dataset — Scraping Project

Dataset collection for Bangla fake news detection using Deep Learning.

## Files
- `scraper.py` — news article scraper (real + fake)
- `youtube_scraper.py` — YouTube channel scraper
- `requirements.txt` — install with: pip install -r requirements.txt

## Setup
```
pip install -r requirements.txt
python scraper.py
python youtube_scraper.py
```

See project_guide.md for full instructions.
